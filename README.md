## 🚀 Quick start (local)

> **Prerequisites**  
> • Node ≥ 18  
> • npm ≥ 9 (or pnpm / yarn if you prefer)

### 1. Front-end

```bash
# from repo root
cd frontend
npm install
npm start      # ➜ http://localhost:3000
```

### 1. Back-end

```bash
# from repo root
cd backend
npm install    # if you have dependencies
node server.js # ➜ http://localhost:5000 (default)
```
